### 移动端特点

```css
/* PC端和移动端的区别 */
PC屏幕大，网页固定版心
手机屏幕小，网页宽度多数为100%

/* 屏幕 */
手机 对角线
PC 宽高: 物理分辨率，逻辑分辨率
参考 逻辑分辨率

/* 百分比布局: 流式布局 */
宽度自适应 高度固定

```

### 移动适配

```css
/* 
1rem = 1HTML字号大小(视口宽度的1/10)
	通过媒体查询设置差异化
*/
@media (width: 375px) {
  html {
    font-size: 37.5px;
  }
}

rem

/*
相对于视口尺寸计算的结果
vw: viewport width
	1vw = 1/100 视口宽度
vh: viewport height
	1vh = 1/100 视口高度
*/
vw / vh
```
