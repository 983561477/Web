### HTML-Emmet语法

```html
<!--
标签名
	快捷方式: div
-->
<div></div>

<!--
类选择器
	快捷方式: .red
-->
<div class="red"></div>

<!--
id选择器
	快捷方式: #one
-->
<div id="one"></div>

<!--
交集选择器
	快捷方式: p.red#one
-->
<p class="red" id="one"></p>

<!--
同级选择器
	快捷方式: div + p
-->
<div></div>
<p></p>

<!--
子代选择器
	快捷方式: ul>li
-->
<ul>
  <li></li>
</ul>

<!--
内部文本
	快捷方式: ul>li{内容}
	数字自增：内容=$ 
-->
<ul>
  <li>内容</li>
</ul>

<!--
创建多个
	快捷方式: ul>li*3
-->
<ul>
  <li></li>
  <li></li>
  <li></li>
</ul>
```

### CSS-Emmet语法

```css
/* 单词首字母 */
/* 宽300 高200 背景色 */
w300+h200+bgc

/* 背景颜色 */
background-color
简写: bgc

/* 
背景图片
	简写
	备注
*/
background-image
简写: bgi
```
