## 快捷键

### HTML骨架

```
! + Enter
```

### 注释

```
ctrl + /
```

### 选择多个重复的值

```
ctrl + d
```

### 复制+换行+粘贴

```
alt + shift + ↓
```

### 替换

```html
crtl + h
```

### 选多行

```html
<!-- 单个 -->
alt
<!-- 多个 -->
alt + shitf
```
